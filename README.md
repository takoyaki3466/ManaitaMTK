警告！
このMODはゲームバランスを崩す可能性があります！
まな板装備の詳細
1: まな板のヘルメットを装備している時、無敵になり飛行可能になります

まな板ツールの詳細
1: まな板パクセルを持った状態で

shift + 右クリック
を押すと破壊範囲を指定することが出来ます。
範囲はノーマル、3×3、5×5、7×7、9×9の順番で入れ替わります

まな板ブロックの詳細
1: まな板ブロックを

shift + 右クリック
すると、増やすサイズを指定することが出来ます
サイズは2^0(1倍)、2^1(2倍)、2^2(4倍)、2^3(8倍)、2^4(16倍)、2^5(32倍)、2^6(64倍)の順で入れ替わります
